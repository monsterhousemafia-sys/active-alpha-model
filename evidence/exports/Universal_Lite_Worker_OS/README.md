# Universal Lite Worker OS

Leicht. Universal. Rechenkraft fürs R3-Netzwerk.

## Schnellstart

| System | Aktion |
|--------|--------|
| Windows | `Windows_START.bat` |
| macOS | `Mac_START.command` |
| Linux | `./Linux_INSTALL.sh` (installiert + startet) |

## Ein-Zeilen-Install (Linux)

```
curl -fsSL http://127.0.0.1:17890/api/ulwo/install.sh | ULWO_HUB=http://127.0.0.1:17890 bash
```

## R3 König

http://127.0.0.1:17890/

Kein Broker. Kein Geld. Nur CPU.
